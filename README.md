## features
* a cute cat whose powers you can control
* a push button activates the motor & LED
* photoresistor: the eyes light up :)

## parts needed
* 1 battery holder
* 1 push button
* 5 LEDs
* 1 photoresistor
* 6 resistors
* 1 vibration motor
